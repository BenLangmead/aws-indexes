# Privacy

## Google Analytics 4

This site may use [Google Analytics 4](https://support.google.com/analytics/answer/10089681) (GA4) to understand aggregate traffic: which pages are viewed, approximate geographic region, and how visitors reach the site. The analytics script runs only if the site operator has enabled a Google Analytics **Measurement ID** in the site configuration.

You can read how Google processes data in [Google’s Privacy & Terms](https://policies.google.com/technologies/partner-sites) and use [Google’s opt-out browser add-on](https://tools.google.com/dlpage/gaoptout) if you wish to reduce data collection across participating sites, or use standard ad/tracker blocking tools.

## This site’s content

The **Index zone** pages list public download locations (HTTPS and S3) for bioinformatics index files. Following an outbound link to download data is between you and the host (e.g. Amazon S3, project websites). This privacy notice covers behavior of the pages hosted on **GitHub Pages** for this project, not third-party destinations.
